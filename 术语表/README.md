# 术语表

> 精简术语卡片库，按一级分类聚合。每条术语包含：定义 + 与相邻术语区分 + 常见误用 + 本仓库相关文档跳转。

## 使用方式

- **快速查询**：直接搜文件内关键词
- **新增术语**：按所属一级分类追加到对应文件，而不是新建独立文件
- **深度阅读**：点击每条术语末尾的"本仓库相关文档"链接

## 术语卡片格式约定

每条术语遵循统一结构：

```markdown
## 术语名（英文全称）

**定义**：一句话定义 + 计算/判定方式（如适用）

**与相邻术语区分**：
- 易混淆术语 1：差异点
- 易混淆术语 2：差异点

**典型量级 / 常见取值**：（可选，数据类术语适用）

**常见误用**：踩坑场景 → 后果

**本仓库相关文档**：
- [文档标题](相对路径)
```

## 维护原则

- **一个术语一段卡片**，不写长篇——长内容沉淀到对应主题目录的文档里
- **术语名 + 英文全称**作为二级标题，便于通过 `## QPS` 类锚点直链
- 当某术语在仓库被反复引用时，把它加进卡片库；只出现一次的临时概念不进
- 新增一级分类时，在本 README 表格中登记
