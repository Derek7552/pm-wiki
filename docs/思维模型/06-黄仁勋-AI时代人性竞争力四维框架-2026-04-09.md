# 黄仁勋：AI时代人性竞争力四维框架

> 来源：笔记侠（Notesman）/ 贾宁 整理自黄仁勋访谈
> 提炼日期：2026-04-09
> 分类：思维模型

## 核心观点

智能正在成为廉价商品——AI能以极低成本完成标准化智力任务（编码、财报分析、方案撰写），与AI在"算力赛道"竞争是错误策略。人类真正的护城河在于AI无法复制的人性能力：韧性、信任、愿景、谦逊。

## 关键知识点

### 1. 韧性：遗忘痛苦的能力

不是"扛住痛苦"，而是**快速遗忘不可控的失败**，把精力放回可控的下一步行动。

面对挫折的三步操作：
1. 拆解问题 → 分清可控/不可控
2. 遗忘不可控的失败、尴尬、羞辱
3. 立刻找一件现在能做的可控小事行动

> 类比：AI学习最重要的属性是"系统性遗忘"（防止过拟合），人也需要遗忘才能重置。

### 2. 信任：超越合同的协作上限

合同只能守住合作底线；信任能拉高合作上限，驱动一群人做没人做过的事。

建立信任的方式：
- 从第一性原理出发，讲清"为什么做这件事"
- 把战略判断毫无保留地分享给合作方
- 给对方充分质疑的机会和足够的尊重

对内：用**公开会议替代一对一会议**——所有问题放台面上，所有人听到上下游诉求，避免信息不对称。

### 3. 愿景：定义"为什么"的能力

AI能帮完成99%的执行，但无法定义"我们为什么要做这件事"、"要解决什么人的什么痛点"。

愿景的作用：给技术和算力找到有意义的方向，在最艰难时保持定力不偏移。

### 4. 谦逊与成长力：永远保持初学者心态

- 不相信"继任计划"：真正关心公司未来的方式，是持续传递知识、赋能每个人，而非培养一个接班人
- 讨厌"持续改进"思维：这种思维默认现在是对的，只能小修小补；应回到零、回到第一性原理重构
- 每次会议都是公开推理会议：摊开思考逻辑、判断依据、顾虑和不确定性，带团队一起推演

## 数据与案例

- 英伟达与台积电合作30年、数百亿美元生意，没有正式合同（基于信任而非约束）
- CUDA决策：将CUDA内置每块GeForce显卡 → 吞掉全部毛利 → 市值从约70亿跌至15亿美元 → 坚持十几年 → 成为AI时代全球计算底座
  > ⚠️ 待验证：70亿→15亿为访谈中的大致数字，非精确财务数据
- Jensen有约60个直属下属，全部是各领域顶尖专家，但从不开一对一会议

## 启发与思考

**对PM的直接映射：**

| 黄仁勋框架 | PM工作场景 |
|----------|-----------|
| 韧性/遗忘 | 需求被砍、项目延期后快速复盘重启，不沉溺于负面情绪 |
| 信任 | 与研发、设计建立透明协作，不靠"需求文档约束"而靠共同理解 |
| 愿景 | 能回答"这个功能为什么存在"，而不只是传递客户需求 |
| 谦逊 | 用"拷问模式"替代"给答案"，带团队一起推演而非单向指令 |

**关于AI时代的工作分类：**
- 80%精力放在"人性价值任务"：与用户聊真实痛点、定长期方向、处理非标问题
- 20%用AI完成"标准化智力任务"：数据整理、模板方案、重复分析

## 原文精华

> "真正的韧性，从来不是扛住痛苦的能力，而是快速遗忘痛苦的能力。"

> "我不相信继任计划，不是因为我是永生的。而是因为如果你真的关心公司在你之后的未来，你今天最该做的事，不是找一个接班人，而是持续地、毫无保留地传递知识、信息、洞察力、技能和经验，赋能你身边的每一个人。"

> "恰恰是一对一会议，才会造成信息不对称。所有问题放在台面上，所有人都能听到上下游的诉求，都能理解彼此的难处，都能一起参与决策，这才是最高效的协作。"
