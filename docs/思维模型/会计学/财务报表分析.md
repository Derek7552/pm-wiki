# 财务报表分析

**核心思想**：评估企业价值

- 资产负债表：财务状况
- 利润表：盈利能力
- 现金流量表：现金管理
- 三张表联动分析

## 1. 资产负债表

**定义：** 反映企业在某一特定时点财务状况的报表，展示企业"拥有什么"和"欠什么"。

### 核心逻辑

**会计恒等式：**
```
资产 = 负债 + 所有者权益
```

这就像一个等式的两边：
- **左边（资产）**：公司拥有的资源
- **右边（负债+权益）**：这些资源的来源

### 基本结构

```
资产负债表（2025年1月4日）

资产                          负债和所有者权益
────────────────────        ────────────────────
流动资产：                   流动负债：
  现金            50,000       应付账款      20,000
  应收账款        30,000       短期借款      30,000
  存货            20,000
                            长期负债：
非流动资产：                   长期借款      50,000
  固定资产       150,000
  无形资产        50,000     所有者权益：
                              股本         100,000
                              未分配利润   100,000
────────────────────        ────────────────────
总资产          300,000     总负债+权益    300,000
```

### 三大组成部分

**1. 资产（左边）- 公司拥有的资源**
- **流动资产**：一年内可变现（现金、应收账款、存货）
- **非流动资产**：长期使用（房产、设备、专利）

**2. 负债（右边上部）- 公司欠别人的钱**
- **流动负债**：一年内要还（应付账款、短期借款）
- **长期负债**：长期债务（长期借款、债券）

**3. 所有者权益（右边下部）- 股东投入的钱 + 赚的钱**
- 股本（股东投资）
- 未分配利润（累计盈利）

### 实际理解

**比喻：** 你买了一套价值 100 万的房子
```
资产：房子价值 100 万
负债：银行贷款 70 万
所有者权益：你的首付 30 万

等式：100万（房子） = 70万（欠银行的） + 30万（你自己的）
```

### 核心价值

- **看清家底**：公司有多少资产
- **评估风险**：负债是否过高
- **衡量净资产**：所有者权益反映真实价值
- **时点快照**：某一天的财务状况

## 2. 利润表

**定义：** 反映企业在一定时期内经营成果的报表，展示企业"赚了多少钱"。

### 核心逻辑

**利润公式：**
```
利润 = 收入 - 费用（成本）
```

**与资产负债表的区别：**
- **资产负债表**：时点报表（某一天的快照）
- **利润表**：时期报表（一段时间的累计）

### 基本结构

```
利润表（2025年1月1日-1月31日）

一、营业收入                          500,000
    减：营业成本                      300,000
        税金及附加                     10,000
        销售费用                       50,000
        管理费用                       40,000
        财务费用                        5,000
────────────────────────────────────
二、营业利润                           95,000

    加：营业外收入                      5,000
    减：营业外支出                      3,000
────────────────────────────────────
三、利润总额                           97,000

    减：所得税费用                     24,250
────────────────────────────────────
四、净利润                             72,750
```

### 关键组成部分

**1. 营业收入**
- 销售商品或提供服务获得的收入
- 公司的主营业务收入

**2. 营业成本**
- 生产商品或提供服务的直接成本
- 原材料、人工、制造费用等

**3. 期间费用**
- **销售费用**：广告、销售人员工资、运输费
- **管理费用**：行政人员工资、办公费、折旧
- **财务费用**：利息支出、手续费

**4. 营业利润**
- 核心业务的盈利能力
- 营业收入 - 营业成本 - 各项费用

**5. 净利润**
- 最终赚到的钱
- 扣除所有成本费用和税后的利润
- 可以分配给股东或留存再投资

### 实际理解

**比喻：开奶茶店一个月的经营**

```
收入：卖奶茶收入          50,000 元
成本：原料成本            20,000 元
费用：
  - 店员工资              8,000 元
  - 房租水电              5,000 元
  - 广告推广              2,000 元
────────────────────────────
营业利润                  15,000 元
税费                       3,000 元
────────────────────────────
净利润                    12,000 元
```

这个月净赚 12,000 元。

### 核心价值

- **评估盈利能力**：企业是否赚钱
- **分析成本结构**：钱花在哪里
- **趋势分析**：对比不同时期利润变化
- **投资决策**：判断企业经营状况

### 与资产负债表的关系

```
利润表的"净利润" → 增加资产负债表的"未分配利润"
```

利润表赚的钱，最终会体现在资产负债表的所有者权益中。

## 3. 现金流量表

**定义：** 反映企业在一定时期内现金流入和流出情况的报表，展示企业"现金怎么流动"。

### 核心问题

**关键矛盾：利润 ≠ 现金**

利润表说你赚了钱，但银行账户可能没钱！为什么？

**例子：**
```
公司卖货赚了 10 万（利润表显示盈利）
但客户说3个月后才付款（应收账款）
银行账户：0 元（没有现金）

结果：账面盈利，但没现金发工资！
```

### 基本结构

```
现金流量表（2025年1月1日-1月31日）

一、经营活动现金流量
    销售商品收到的现金               450,000
    收到的其他现金                    10,000
    购买商品支付的现金              -280,000
    支付给员工的现金                 -80,000
    支付的各项税费                   -30,000
    ────────────────────────────
    经营活动现金流量净额               70,000

二、投资活动现金流量
    购建固定资产支付的现金          -100,000
    处置设备收到的现金                20,000
    ────────────────────────────
    投资活动现金流量净额              -80,000

三、筹资活动现金流量
    吸收投资收到的现金               100,000
    偿还借款支付的现金               -50,000
    支付利息                          -5,000
    ────────────────────────────
    筹资活动现金流量净额               45,000

────────────────────────────────
现金净增加额                          35,000

期初现金余额                          50,000
期末现金余额                          85,000
```

### 三大活动

**1. 经营活动现金流（最重要）**

日常经营产生的现金流

流入：
- 销售商品收到的现金
- 提供服务收到的现金

流出：
- 购买原材料支付的现金
- 支付员工工资
- 缴纳税费

**核心指标：** 经营活动现金流为正 = 主业能赚到真金白银

**2. 投资活动现金流**

买卖资产的现金流

流出（通常为负）：
- 购买设备、厂房
- 投资其他公司

流入：
- 出售资产
- 收回投资

**正常企业：** 投资活动现金流为负（扩张投资）

**3. 筹资活动现金流**

融资的现金流

流入：
- 股东投资
- 银行借款

流出：
- 偿还借款
- 分红给股东

### 实际理解

**比喻：你的个人账户一个月**

```
经营活动（工作赚钱）：
  工资收入            +15,000
  日常消费            -8,000
  ──────────────
  经营活动现金流      +7,000 ✓（赚到现金）

投资活动（买资产）：
  买股票              -10,000
  ──────────────
  投资活动现金流      -10,000 （正常，投资未来）

筹资活动（借钱还钱）：
  父母给钱            +5,000
  还信用卡            -2,000
  ──────────────
  筹资活动现金流      +3,000

──────────────────────
现金净增加              0
```

### 与利润表的关系

**关键差异：**

| 项目 | 利润表 | 现金流量表 |
|------|--------|-----------|
| 销售收入 | 10万（开发票就算） | 8万（实际收到现金） |
| 购买设备 | 0（资产不算费用） | -10万（支付现金） |
| 折旧 | -2万（费用） | 0（不涉及现金） |

**例子：**
```
利润表：净利润 5 万（看起来赚钱）
现金流量表：现金流 -3 万（实际缺钱）

原因：
- 客户欠款未付（应收账款）
- 购买了设备（利润表不体现，现金流体现）
```

### 核心价值

- **现金为王**：评估企业真实的现金状况
- **预警功能**：利润好但现金流差 = 危险信号
- **生存能力**：有现金才能活下去
- **质量评估**：经营活动现金流 > 净利润 = 盈利质量高

### 健康企业的特征

```
✓ 经营活动现金流：正值且大于净利润
✓ 投资活动现金流：负值（持续投资未来）
✓ 筹资活动现金流：根据阶段不同
```

## 三张表联动分析

**完整的财务画像需要三张表结合：**

1. **资产负债表** - 看家底（有多少资产、负债）
2. **利润表** - 看盈利（赚了多少钱）
3. **现金流量表** - 看现金（真实的现金状况）

**关键关系：**
```
利润表的净利润 → 资产负债表的未分配利润（所有者权益）
现金流量表的现金变化 → 资产负债表的现金余额
```

**健康企业的标志：**
- 资产 > 负债（资产负债表）
- 净利润为正（利润表）
- 经营现金流为正且 > 净利润（现金流量表）
