# 认知偏见

**核心思想**：避免思维陷阱

---

## 一、常见认知偏见

### 1. 确认偏误（Confirmation Bias）

倾向于寻找支持自己观点的证据，忽略反对证据。

**例子：**
```
你认为某只股票会涨：
✗ 只看利好消息，忽略风险提示
✗ 只关注成功案例，忽略失败案例
✓ 主动寻找反对意见，全面评估
```

---

### 2. 锚定效应（Anchoring Effect）

过度依赖第一印象或最初信息。

**例子1：商品定价**
```
商品原价：999元
促销价：499元（感觉很便宜）

实际成本可能只有100元，但被原价"锚定"了判断
```

**例子2：薪资谈判**
```
HR先开价：月薪8000
你被锚定，最后谈到9000就满意了

如果你先开价：月薪15000
HR被锚定，最后谈到12000
```

---

### 3. 幸存者偏差（Survivorship Bias）

只看到成功者，忽略失败者。

**例子：**
```
看到辍学创业成功的比尔·盖茨、扎克伯格
结论：辍学创业能成功 ✗

没看到：数百万辍学失败的人
正确认知：成功是小概率事件，需要全面评估
```

---

### 4. 沉没成本谬误（Sunk Cost Fallacy）

因为已经投入而继续错误决策。

**例子：**
```
已投入项目100万，发现方向错了
✗ "已经花了100万，不能放弃"（追加投入）
✓ "未来还会亏多少？"（止损）

过去的成本已沉没，不应影响未来决策
```

---

### 5. 可得性偏差（Availability Bias）

容易回忆的事件被高估发生概率。

**例子：**
```
新闻报道飞机失事：
→ 觉得飞机很危险
实际：飞机是最安全的交通工具

因为飞机事故新闻报道多，容易记住
```

---

## 二、认知偏见汇总表

| 偏见 | 表现 | 危害 |
|------|------|------|
| **确认偏误** | 只看支持自己的证据 | 决策片面 |
| **锚定效应** | 被第一个数字影响 | 判断失准 |
| **幸存者偏差** | 只看成功案例 | 高估成功率 |
| **沉没成本** | 因已投入而坚持 | 越陷越深 |
| **可得性偏差** | 印象深的被高估 | 概率误判 |

---

## 三、应对策略

| 策略 | 说明 |
|------|------|
| **主动寻找反对意见** | 强迫自己看另一面 |
| **使用清单和流程** | 避免直觉判断 |
| **延迟决策** | 不在情绪激动时决策 |
| **咨询他人** | 获得不同视角 |
| **记录决策过程** | 事后复盘改进 |

---

## 四、核心启示

| 启示 | 说明 |
|------|------|
| **质疑直觉** | 大脑存在系统性偏差 |
| **建立流程** | 用流程对抗偏见 |
| **保持开放** | 接受与自己相反的观点 |
| **持续学习** | 了解更多认知偏见 |

---

> **一句话总结**：认知偏见是大脑的"系统Bug"，知道它们存在，才能主动避开陷阱。
