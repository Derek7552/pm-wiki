# 机会成本

**核心思想**：选择的代价

---

## 一、基本概念

**机会成本（Opportunity Cost）**：做某件事的真正成本，是放弃的最佳替代选择。

**核心认知：**
- 选择A = 放弃B + 放弃C + ...
- 真正的成本不是支付的金钱，而是失去的机会
- 时间是最稀缺的资源

---

## 二、实际例子

### 例子1：上大学的机会成本

```
显性成本：
学费：4年×1万 = 4万元

机会成本：
4年本可以工作赚钱：4年×5万/年 = 20万元

真正成本 = 4万 + 20万 = 24万元

是否值得？
- 看未来收入提升
- 看职业发展机会
- 综合评估
```

### 例子2：创业 vs 打工

```
打工：
年薪：50万（确定）
机会成本：创业可能的收益

创业：
预期收入：可能100万，也可能0
机会成本：放弃的50万稳定收入

决策：
评估创业成功概率和预期收益
是否值得放弃50万
```

### 例子3：看一部电影

```
显性成本：
电影票：50元
时间：2小时

机会成本：
2小时本可以：
- 阅读一本书的1/3
- 学习一个技能
- 陪伴家人
- 工作赚钱（假设时薪100，就是200元）

真正成本：50元 + 200元 = 250元

值得吗？看电影带来的价值是否 > 250元
```

### 例子4：免费的东西

```
"免费"讲座：
金钱成本：0元
时间成本：3小时

机会成本：
3小时本可以做更有价值的事

启示：
- 免费不代表没有成本
- 时间成本往往更高
- 要评估时间价值
```

### 例子5：投资决策

```
投资A：年化收益10%
投资B：年化收益15%

选择A的机会成本：
放弃B的5%收益差

100万投资1年：
选A赚10万，但机会成本是5万
实际收益 = 10万 - 5万 = 5万（相对于B）
```

## 三、时间的机会成本

### 时薪思维

```
假设年收入50万：
工作日250天，每天8小时
时薪 = 50万 / (250×8) = 250元/小时

评估任何活动：
- 花1小时做家务：机会成本250元
  → 是否请保洁（100元/小时）更划算？

- 花2小时通勤：机会成本500元
  → 是否租靠近公司的房子更划算？

- 花3小时排队买便宜货：机会成本750元
  → 省了50元，亏了700元
```

### 高价值人群的时间成本

```
马云的时薪可能是几万元
→ 他绝不会花时间做低价值的事
→ 时间全用在高杠杆活动上

普通人也应该：
- 计算自己的时薪
- 避免低价值活动
- 专注高价值工作
```

## 四、沉没成本 vs 机会成本

| | 沉没成本 | 机会成本 |
|---|----------|----------|
| **定义** | 已经发生、无法收回的成本 | 未来的选择成本 |
| **应该影响决策吗？** | ❌ 不应该 | ✅ 应该 |

### 例子：烂电影

```
情况：
花50元买票，看了30分钟发现很烂

错误思维（沉没成本谬误）：
"已经花了50元和30分钟，不看完太亏了"
→ 继续看1.5小时

正确思维（机会成本）：
"50元和30分钟已经沉没了
继续看1.5小时的机会成本是什么？"
→ 离开，做更有价值的事

启示：
- 沉没成本不应影响未来决策
- 只考虑未来的机会成本
```

## 五、隐性机会成本

### 例子1：自己做 vs 外包

```
自己学习财务记账：
学习成本：50小时
每月记账：5小时
一年：60小时

请代账公司：
费用：每年5000元

机会成本评估：
60小时能创造多少价值？
如果 > 5000元，就应该外包
```

### 例子2：完美主义

```
做PPT：
80分方案：2小时
95分方案：10小时

机会成本：
8小时本可以做什么？
- 做4个80分方案
- 学习新技能

启示：
完美主义的机会成本很高
80分原则更高效
```

## 六、应用策略

### 个人层面

```
✓ 计算时薪，重视时间成本
✓ 外包低价值工作
✓ 专注高价值活动
✓ 评估每个选择的机会成本
✓ 学会说"不"
```

### 工作层面

```
✓ 优先做高价值工作
✓ 自动化重复性任务
✓ 委托团队成员
✓ 避免完美主义
```

### 投资层面

```
✓ 评估不同投资的机会成本
✓ 资产配置考虑机会成本
✓ 现金也有机会成本（通货膨胀）
```

---

## 七、核心启示

| 启示 | 说明 |
|------|------|
| **时间最宝贵** | 时间机会成本往往被忽视 |
| **真正的成本是机会** | 不是花了多少钱，而是失去了什么 |
| **忽略沉没成本** | 过去的不应影响未来 |
| **理性决策** | 每个选择都要算机会成本 |

---

> **一句话总结**：选择的真正代价不是你付出了什么，而是你放弃了什么。时间是最大的机会成本。
