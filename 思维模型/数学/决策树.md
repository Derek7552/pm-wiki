# 决策树

**核心思想**：理清复杂决策

---

## 一、基本结构

```
          决策点
         /      \
    选择A        选择B
      |            |
   结果1         结果2
   (概率50%)    (概率50%)
   收益10万     收益5万
```

**组成要素**：
- **决策节点**（方框）：你要做的选择
- **机会节点**（圆圈）：不确定的结果
- **分支**：可能的选项或结果
- **概率**：每个结果发生的可能性
- **收益**：每个结果的价值

---

## 二、实际例子

### 例子1：创业决策

```
                要不要创业？
               /          \
           创业              不创业
          /    \               |
      成功(30%)  失败(70%)    上班
         |         |           |
      年入100万  年入10万    年入40万

创业期望值 = 0.3×100 + 0.7×10 = 30 + 7 = 37万
不创业 = 40万

理性选择：不创业（但要考虑非货币因素）
```

### 例子2：产品方向选择

```
                新产品方向
               /          \
          ToB方向         ToC方向
          /    \          /    \
      成功(40%) 失败    成功(20%) 失败
         |       |        |       |
      1000万   -100万   5000万   -200万

ToB期望值 = 0.4×1000 + 0.6×(-100) = 400 - 60 = 340万
ToC期望值 = 0.2×5000 + 0.8×(-200) = 1000 - 160 = 840万

期望值：ToC更高（但风险也更大）
```

### 例子3：职业发展

```
              现在30岁，如何发展？
            /         |         \
        深耕技术    转管理      创业
         /  \       /  \       /  \
    成功  失败   成功  失败   成功  失败
     |     |     |     |     |     |
   年薪   年薪   年薪   年薪  财富  年薪
   80万  50万  100万  40万  500万 20万

需要评估：
- 各路径成功概率
- 个人能力匹配度
- 风险承受能力
```

---

## 三、应用场景

| 场景 | 应用 |
|------|------|
| **战略规划** | 评估不同战略路径 |
| **投资决策** | 分析投资方案 |
| **职业规划** | 选择职业路径 |
| **项目管理** | 评估项目风险 |

---

## 四、核心价值

| 价值 | 说明 |
|------|------|
| **可视化决策** | 清晰看到所有选项 |
| **逻辑梳理** | 理清因果关系 |
| **量化对比** | 数据化比较方案 |
| **风险识别** | 提前发现潜在风险 |

---

## 五、使用步骤

```
1. 列出选项    → 所有可能的选择
       ↓
2. 画出分支    → 每个选择的可能结果
       ↓
3. 标注概率    → 每个结果的发生概率
       ↓
4. 计算期望值  → 结合概率计算收益
       ↓
5. 做出决策    → 选择期望值最高的路径
```

---

## 六、注意事项

| 注意点 | 说明 |
|--------|------|
| **概率估计要准确** | 垃圾进，垃圾出 |
| **考虑非货币因素** | 成就感、健康、家庭 |
| **风险承受能力** | 期望值高不代表适合你 |
| **动态调整** | 情况变化时重新评估 |

---

> **一句话总结**：把复杂决策画成树状图，用概率和期望值量化比较，让决策变得清晰理性。
