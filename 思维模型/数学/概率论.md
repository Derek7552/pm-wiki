# 概率论

**核心思想**：评估风险和收益

---

## 一、基本概念

### 概率

事件发生的可能性（0-1之间）

| 概率值 | 含义 |
|--------|------|
| 0 | 不可能发生 |
| 0.5 | 50%的可能性 |
| 1 | 必然发生 |

### 期望值

长期平均收益

```
期望值 = Σ(结果 × 概率)
```

---

## 二、实际例子

### 例子1：投资决策

```
方案A：
- 70%概率赚10万
- 30%概率亏2万
期望值 = 0.7×10 + 0.3×(-2) = 7 - 0.6 = 6.4万

方案B：
- 50%概率赚8万
- 50%概率亏1万
期望值 = 0.5×8 + 0.5×(-1) = 4 - 0.5 = 3.5万

✓ 选择方案A（期望值更高）
```

### 例子2：求职选择

```
大厂offer：
- 80%概率稳定发展，年薪40万
- 20%概率被裁员，收入0
期望值 = 0.8×40 + 0.2×0 = 32万

创业公司：
- 30%概率成功，年薪80万
- 70%概率失败，年薪20万
期望值 = 0.3×80 + 0.7×20 = 24 + 14 = 38万

✓ 根据期望值，创业公司更优（如果概率估计准确）
```

### 例子3：产品上线

```
快速上线：
- 60%概率成功，收益100万
- 40%概率失败，损失20万
期望值 = 0.6×100 + 0.4×(-20) = 60 - 8 = 52万

充分测试后上线：
- 90%概率成功，收益80万
- 10%概率失败，损失5万
期望值 = 0.9×80 + 0.1×(-5) = 72 - 0.5 = 71.5万

✓ 充分测试策略更优
```

## 三、应用场景

| 场景 | 应用 |
|------|------|
| **投资决策** | 计算预期收益 |
| **风险管理** | 量化风险大小 |
| **产品迭代** | 评估功能优先级 |
| **保险购买** | 评估是否值得 |

---

## 四、核心启示

| 启示 | 说明 |
|------|------|
| **不确定性量化** | 用概率表达不确定性 |
| **长期思维** | 单次结果不重要，期望值才重要 |
| **理性决策** | 基于概率而非直觉 |
| **风险平衡** | 高收益往往伴随高风险 |

---

## 五、常见误区

| 误区 | 正确理解 |
|------|----------|
| 赌徒谬误 | 前一次结果不影响下一次概率 |
| 忽视基础概率 | 要考虑事件本身发生的概率 |
| 过度自信 | 人们常高估自己成功的概率 |

---

> **一句话总结**：用概率思考世界，用期望值做决策。单次输赢不重要，长期期望值为正才重要。
