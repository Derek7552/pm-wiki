# 排列组合

**核心思想**：分析可能性

---

## 一、基本概念

### 排列（有顺序）

A、B、C 三个人排队，有多少种排法？

```
排法数 = 3 × 2 × 1 = 6 种
ABC、ACB、BAC、BCA、CAB、CBA
```

### 组合（无顺序）

A、B、C 三个人选2个，有多少种选法？

```
选法数 = 3 种
AB、AC、BC
（顺序不重要，AB和BA是同一种）
```

### 区别

| | 排列 | 组合 |
|---|------|------|
| **顺序** | 重要 | 不重要 |
| **例子** | 密码、排名 | 选人、配菜 |
| **公式** | P(n,r) = n!/(n-r)! | C(n,r) = n!/[r!(n-r)!] |

---

## 二、实际例子

### 例子1：职业选择

```
行业选择：3个（科技、金融、制造）
职位类型：4个（技术、销售、管理、运营）
工作地点：5个城市

总可能性 = 3 × 4 × 5 = 60 种组合
```

**启示**：在做选择前，先算清楚有多少种可能

### 例子2：密码安全

```
4位数字密码：10^4 = 10,000 种可能
6位数字密码：10^6 = 1,000,000 种可能

增加2位，安全性提升100倍
```

### 例子3：投资组合

```
有10只股票，选3只构建组合
组合数 = C(10,3) = 120 种

系统评估所有可能，找到最优组合
```

---

## 三、应用场景

| 场景 | 应用 |
|------|------|
| **决策分析** | 列举所有可能方案 |
| **风险评估** | 计算不同情况的数量 |
| **产品设计** | 功能组合的可能性 |
| **AB测试** | 多变量测试的组合数 |

---

## 四、核心启示

| 启示 | 说明 |
|------|------|
| **全面思考** | 不遗漏任何可能性 |
| **量化评估** | 用数字衡量选择空间 |
| **避免遗漏** | 系统化列举，减少盲点 |

---

> **一句话总结**：先算清楚有多少种可能，再做选择。排列组合帮你看到全貌。
