# 复式记账

## 一、什么是复式记账？

**复式记账**是一种记录经济活动的方法：**每笔交易同时记录在两个或以上的账户中，借贷双方金额相等。**

> 核心理念：钱不会凭空产生，也不会凭空消失。每一笔资金都有**来源**和**去向**。

---

## 二、首先澄清：借和贷是什么？

### ⚠️ 最重要的概念

**借、贷只是账本的左右两边，不是两个人！**

```
            公司账本
        ┌─────────────┐
        │  借方 │ 贷方 │
        │  左边 │ 右边 │
        └─────────────┘
```

- **借（Debit）** = 账本**左边**
- **贷（Credit）** = 账本**右边**
- 都是从**同一个会计主体（公司）**的角度记录

### ❌ 常见误解

| 错误理解 | 正确理解 |
|---------|---------|
| 借 = 借入、流入 | 借 = 记在左边 |
| 贷 = 贷出、流出 | 贷 = 记在右边 |
| 借方是别人 | 借贷都是记录自己公司的账 |

---

## 三、理论基础：会计恒等式

复式记账建立在这个恒等式上：

```
资产 = 负债 + 所有者权益
```

展开后：

```
资产 = 负债 + 所有者权益 + (收入 - 费用)
```

**为什么借贷必相等？**

因为每笔交易都同时影响等式的两边（或同一边的两个项目），等式始终保持平衡。

---

## 四、核心规则：什么时候记借方/贷方？

### 借贷记账规则表

| 账户类型 | 借方（左）| 贷方（右）|
|---------|----------|----------|
| **资产** | ➕ 增加 | ➖ 减少 |
| **负债** | ➖ 减少 | ➕ 增加 |
| **所有者权益** | ➖ 减少 | ➕ 增加 |
| **收入** | ➖ 冲减 | ➕ 增加 |
| **费用** | ➕ 增加 | ➖ 冲减 |

### 记忆口诀

```
资产费用借方增，负债权益贷方增
收入增加记贷方，费用增加记借方
```

### 为什么是这样？

从会计恒等式理解：

```
资产 = 负债 + 所有者权益

左边        右边
```

- **资产**在等式左边 → 增加记**借方（左）**
- **负债、权益**在等式右边 → 增加记**贷方（右）**
- **收入**增加权益 → 记贷方
- **费用**减少权益 → 记借方（与权益相反）

## 五、实战示例

### 示例1：购买电脑 10,000 元（现金支付）

**分析**：
- 获得电脑（固定资产↑）
- 支出现金（银行存款↓）
- 两个都是资产，一增一减

```
借：固定资产（电脑）  10,000 元  ← 资产增加，记借方
贷：银行存款          10,000 元  ← 资产减少，记贷方
```

✅ 借贷相等，平衡！

---

### 示例2：银行贷款 50,000 元

**分析**：
- 银行存款增加（资产↑）
- 欠银行的钱增加（负债↑）

```
借：银行存款    50,000 元  ← 资产增加，记借方
贷：银行借款    50,000 元  ← 负债增加，记贷方
```

⚠️ 注意：钱流入了，但不是记在"贷方"，而是根据账户性质（资产增加）记在借方！

---

### 示例3：偿还贷款 10,000 元

**分析**：
- 欠款减少（负债↓）
- 银行存款减少（资产↓）

```
借：银行借款    10,000 元  ← 负债减少，记借方
贷：银行存款    10,000 元  ← 资产减少，记贷方
```

---

### 示例4：销售商品收入 8,000 元

**分析**：
- 银行收到钱（资产↑）
- 产生销售收入（收入↑）

```
借：银行存款    8,000 元  ← 资产增加，记借方
贷：销售收入    8,000 元  ← 收入增加，记贷方
```

---

### 示例5：支付员工工资 5,000 元

**分析**：
- 银行存款减少（资产↓）
- 产生工资费用（费用↑）

```
借：工资费用    5,000 元  ← 费用增加，记借方
贷：银行存款    5,000 元  ← 资产减少，记贷方
```

## 六、复式记账的价值

### 1. 自动纠错

借贷不相等 = 一定记错了。这是一个**内置的校验机制**。

### 2. 资金流向清晰

每笔交易都回答两个问题：
- **钱从哪来？**（贷方来源）
- **钱到哪去？**（借方去向）

### 3. 系统性思维

> 芒格在《穷查理宝典》中将复式记账列为重要的思维模型：它教会我们看到事物的**两面性**——任何经济活动都有对应的另一面。

---

## 七、快速判断流程图

```
遇到一笔交易时：

1. 识别涉及哪些账户？
   ↓
2. 每个账户是什么类型？（资产/负债/权益/收入/费用）
   ↓
3. 该账户是增加还是减少？
   ↓
4. 查规则表：
   - 资产/费用 增加 → 借方
   - 资产/费用 减少 → 贷方
   - 负债/权益/收入 增加 → 贷方
   - 负债/权益/收入 减少 → 借方
   ↓
5. 检查：借方合计 = 贷方合计？
```

## 八、总结

| 要点 | 说明 |
|------|------|
| **本质** | 每笔交易记录两次，借贷相等 |
| **借贷** | 只是左右方向，不是流入流出 |
| **依据** | 会计恒等式：资产 = 负债 + 权益 |
| **价值** | 纠错、追踪资金、系统思维 |

> **一句话总结**：复式记账用"借"和"贷"两个方向，记录每笔交易的来龙去脉，确保账目永远平衡。
